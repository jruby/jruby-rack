Getting started

Here’s how to get rolling:

  1. Download and install JRuby

    Grab a tarball, and just unpack and run!

  2. Install warbler

    jruby -S gem install warbler

  3. Use this sample application as a starting point

    You can get it from JRuby-Rack's repository on Github.

  4. Change the application id

    Edit appengine-web.xml and replace it in the <application> section.

  5. Deploy!

    jruby -S rake appengine:deploy

  6. Join the JRuby community

    Join the mailing lists to stay on top of the latest developments with Rails on Google App Engine.
